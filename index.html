Domowe Opłaty - Dokumentacja

Witamy w dokumentacji aplikacji Domowe Opłaty - kompleksowego narzędzia do zarządzania budżetem domowym.

O programie

Domowe Opłaty to zaawansowana aplikacja desktopowa stworzona w Python z wykorzystaniem PyQt5, zaprojektowana z myślą o efektywnym zarządzaniu finansami domowymi.

Główne funkcje:

  • Śledzenie wydatków i przychodów
  • Zarządzanie cyklicznymi płatnościami
  • Generowanie raportów i wykresów
  • System powiadomień o zbliżających się płatnościach
  • Wielojęzyczny interfejs
  • Automatyczne kopie zapasowe

Instalacja

Wymagania systemowe:

  • System operacyjny: Windows 10/11, Linux, macOS
  • Python 3.8 lub nowszy
  • Minimum 2GB RAM
  • 100MB wolnego miejsca na dysku

Proces instalacji:

# 1. Instalacja przez pip
pip install domowe-oplaty

# 2. Lub klonowanie repozytorium
git clone https://github.com/twoj-username/domowe-oplaty.git
cd domowe-oplaty
pip install -r requirements.txt
Porada: Zalecamy utworzenie wirtualnego środowiska Python przed instalacją.

Pierwsze kroki

Po zainstalowaniu aplikacji, wykonaj następujące kroki:

  1. Uruchom aplikację poprzez komendę: domowe-oplaty
  2. Przy pierwszym uruchomieniu zostaniesz poproszony o:
    • Wybór języka interfejsu
    • Ustawienie hasła dostępu (opcjonalne)
    • Konfigurację podstawowych kategorii wydatków

Konfiguracja

Aplikacja oferuje szereg opcji konfiguracyjnych dostępnych w menu Ustawienia:

Podstawowa konfiguracja

# Przykładowy plik konfiguracyjny config.ini
[General]
Language = pl
AutoBackup = true
BackupInterval = 7
NotificationsEnabled = true

[Database]
Path = ./data/expenses.db
AutoVacuum = true
Uwaga: Po zmianie ustawień konieczne jest ponowne uruchomienie aplikacji.

Zarządzanie wydatkami

Dodawanie wydatków

Aby dodać nowy wydatek:

  1. Kliknij przycisk "Nowy wydatek"
  2. Wypełnij formularz:
    • Kwota
    • Data
    • Kategoria
    • Opis (opcjonalnie)
  3. Zatwierdź przyciskiem "Zapisz"

Raporty i analizy

System raportowania pozwala na generowanie różnych typów zestawień:

  • Raporty miesięczne
  • Zestawienia roczne
  • Analizy kategorii
  • Prognozy wydatków

Kopie zapasowe

System automatycznie tworzy kopie zapasowe:

  • Codzienny backup danych
  • Przechowywanie 7 ostatnich kopii
  • Możliwość ręcznego tworzenia kopii

FAQ

Najczęściej zadawane pytania

P: Jak zmienić język aplikacji?

O: Przejdź do Ustawienia -> Język i wybierz preferowany język z listy.

P: Czy można importować dane z innych aplikacji?

O: Tak, aplikacja wspiera import danych w formacie CSV i XML.

P: Jak często tworzone są kopie zapasowe?

O: Domyślnie co 7 dni, ale można zmienić tę wartość w ustawieniach.
