Domowe Opłaty - Dokumentacja

Aplikacja do zarządzania domowymi opłatami i budżetem.

O programie

Program służy do efektywnego zarządzania wydatkami domowymi, śledzenia opłat i planowania budżetu.

Instalacja

Wymagania systemowe:

  • Python 3.8+
  • PyQt5
  • SQLite3

Instalacja przez pip:

pip install domowe-oplaty
